# Phone-book
(краткое описани возможностей кода)Что умеет:

Добавлять контакты (имя + номер)

Искать номер по имени

Удалять контакты

Показывать всю книгу

Особенности:

Поиск не зависит от регистра (можно вводить "ПЕТЯ", "петя", "Петя")

Есть несколько контактов при запуске

Данные хранятся только в памяти (не сохраняются в файл)
